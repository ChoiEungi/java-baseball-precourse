# 미션 - 숫자 야구 게임

---

## 🚀 기능(역할) 요구사항

- 입력을 받는 역할

  - 입력이 유효한지 확인하는 역할
    - 1-9로 이루어진 숫자인지
    - 입력 길이가 3인지
    - 각 숫자의 내용이 다른지.
  - 입력을 게임을 진행 하는 역할에 넘겨주는 역할
  


- 게임을 진행 하는 역할

  - 컴퓨터의 수(랜덤한 수)를 만드는 역할
  - 결과를 나타내는(계산하는) 역할
    - 볼의 개수를 계산하는 역할
    - 스트라이크의 개수를 계산하는 역할
    
  
  
  
- 출력을 하는 역할
  
  
  - 결과를 출력해주는 역할
  
    - 스트라이크, 볼을 보고 그에 맞는 결과를 출력을 해준다.
    - 낫띵인지 판단해주는 역할
  
  - 게임을 재시작할지 정하는 역할
